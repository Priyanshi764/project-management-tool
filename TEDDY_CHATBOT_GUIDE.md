## 🗣️ What Teddy Can Help With

### 1. Creating Tasks
**Ask**: "How do I create a task?"

**Teddy says**:
```
To create a new task, you can:

1. Click the '➕ Add Task' button in the header
2. Or press Ctrl+N (keyboard shortcut)
3. Or use a template by clicking '📋 Templates'

Would you like me to explain templates? 🎯
```

### 2. Using Templates
**Ask**: "Tell me about templates"

**Teddy says**:
```
Templates are awesome! 📋 They save you time by pre-filling tasks with:

• Checklists
• Labels
• Priority levels

Click the '📋 Templates' button and choose from:
🐛 Bug Fix
✨ New Feature
👀 Code Review
📚 Documentation
🎨 Design Task
📅 Meeting Prep

Try one now! 🚀
```

### 3. Pomodoro Timer
**Ask**: "How does the Pomodoro timer work?"

**Teddy says**:
```
The Pomodoro Timer helps you focus! ⏱️

1. Click the '⏱️ Pomodoro' button
2. Choose Focus (25 min) or Break (5 min)
3. Click Start and work without distractions!

After 4 focus sessions, take a long break. You've got this! 💪
```

### 4. Keyboard Shortcuts
**Ask**: "What are the keyboard shortcuts?"

**Teddy says**:
```
Here are the keyboard shortcuts! ⚡

• Ctrl+N - Create task
• Ctrl+K - Search
• Ctrl+F - Toggle filters
• Ctrl+A - View activity
• Ctrl+P - Pomodoro timer
• Ctrl+T - Templates
• Ctrl+Space - Quick actions

You're becoming a power user! 🎮
```

### 5. Filters & Search
**Ask**: "How do I use filters?"

**Teddy says**:
```
Filters help you find tasks quickly! 🔍

1. Click the 'Filters' button
2. Use the search box to find tasks
3. Filter by priority (High/Medium/Low)
4. Filter by assignee

You can combine multiple filters too! 🎯
```

### 6. Mood Tracking
**Ask**: "Tell me about mood tracking"

**Teddy says**:
```
I care about how you're feeling! 💭

Click the mood button to set:
🚀 Excited - You're pumped!
💪 Confident - You got this!
😐 Neutral - Just working
😟 Concerned - Need some help?
🚫 Blocked - Let's get you unstuck!

Your team can see when you need support! 🤗
```

### 7. Activity Feed
**Ask**: "What is the activity feed?"

**Teddy says**:
```
The Activity Feed shows everything happening! 📊

Click the purple 'Activity' button to see:
• Task movements
• Status changes
• Comments
• Who did what and when

Stay in the loop! 👀
```

### 8. Moving Tasks
**Ask**: "How do I move tasks between columns?"

**Teddy says**:
```
Moving tasks is easy! 📋

Each task has a dropdown at the bottom:
• To Do - Not started
• In Progress - Working on it
• Done - Completed!

Just click the dropdown and select the new status. The task moves instantly! ✨
```

## 🐛 Troubleshooting

### Can't see Teddy?
- Refresh the page (Ctrl+R or Cmd+R)
- Check bottom left corner
- Make sure you're on Dashboard or Project page

### Chat not opening?
- Click directly on the teddy face
- Try refreshing the page
- Check browser console for errors

### Teddy not responding?
- Check your internet connection
- Refresh the page
- Try asking a different question
